Introdução a Tipos de dados

Em programação, nem toda informação é igual. Às vezes guardamos um número, outras vezes uma palavra ou uma resposta como sim ou não. Para que o computador entenda, usamos os tipos de dados, que classificam as informações e ajudam o programa a lidar corretamente com cada valor.

Os tipos primitivos são os mais básicos. O int representa números inteiros, como 5 ou -2. Para números decimais, usamos float, como 3.14 ou -0.5. Para textos, usamos string, como "Olá mundo" ou "Pacman". Algumas linguagens também têm char, para um único caractere, como 'A'.

Outro tipo importante é o bool, que representa valores booleanos: verdadeiro ou falso. Esse tipo é muito usado em decisões no código. Por exemplo: jogoAtivo = verdadeiro indica que o jogo está em andamento, e jogoAtivo = falso que ele foi pausado ou terminou.

Entender os tipos de dados é essencial para escrever códigos corretos e eficientes. Usar o tipo certo evita erros e facilita o trabalho do computador. Sempre pense qual tipo de informação está lidando e escolha o mais adequado.
