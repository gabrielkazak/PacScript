Introdução a Constantes

Em programação, constantes são como variáveis, mas com uma diferença essencial: seu valor não pode ser alterado após ser definido. Pense nelas como etiquetas permanentes — uma vez colada, não dá para mudar.

Usamos constantes quando queremos garantir que uma informação permaneça estável. Por exemplo: se o número máximo de vidas de um jogador é 5, criamos uma constante MAX_VIDAS = 5. Assim, o programa sempre saberá esse limite, evitando modificações acidentais.

Para criar uma constante, geralmente usamos letras maiúsculas, como VALOR_DO_PONTO ou VELOCIDADE_MÁXIMA. O valor atribuído pode ser de qualquer tipo de dado (int, float, bool, string). Depois disso, tentar alterar gera erro. Isso torna o código mais seguro e organizado, especialmente em projetos grandes.

Usar constantes é importante para escrever códigos mais confiáveis. Elas indicam claramente o que não deve ser alterado, facilitando a manutenção e evolução do programa.
