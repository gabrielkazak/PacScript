Introdução a operadores de comparação

Em programação, operadores de comparação são essenciais para tomar decisões com base em condições. Eles comparam dois valores e retornam um valor booleano: true (verdadeiro) ou false (falso). Os principais operadores de comparação são:

== (igualdade), != (diferença)
> (maior que), < (menor que)
>= (maior ou igual a), <= (menor ou igual a)

Por exemplo, imagine que queremos verificar se o jogador ganhou o jogo: se (pontuacao >= 100), ou seja, "Se a pontuação for maior ou igual a 100, então...". Já para verificar se ele perdeu todas as vidas: se (vidas == 0). Também podemos usar != para verificar se algo é diferente, como: se (cor != "vermelho"), interpretado como "se a cor não for vermelha...".

Esses operadores são úteis quando queremos executar comandos somente se uma condição for verdadeira, como mostrar uma mensagem, parar o jogo ou liberar um novo nível. É importante lembrar: ao comparar valores, estamos perguntando algo ao programa, e a resposta será sempre sim ou não (true ou false). Por isso, eles não mudam valores, apenas avaliam e retornam um resultado booleano.
