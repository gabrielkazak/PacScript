Introdução a Operadores Lógicos

Em programação, às vezes precisamos tomar decisões com base em mais de uma condição. Para isso, usamos os operadores lógicos, que ajudam a combinar comparações e criar regras mais complexas. Eles são como palavras que o computador entende para dizer “e”, “ou” e “não”.

O operador && (E) verifica se duas ou mais condições são verdadeiras ao mesmo tempo. Por exemplo: vidas > 0 && pontosColetados < pontosTotais. O resultado só será verdadeiro se ambas as condições forem verdadeiras.

Já o operador || (OU) é usado quando basta que uma das condições seja verdadeira. Por exemplo: vidas > 0 || temPowerUp == true. Se qualquer uma das duas for verdadeira, o resultado será verdadeiro.

O operador ! (NÃO) serve para inverter um valor lógico: se algo é verdadeiro, o ! transforma em falso, e vice-versa. Por exemplo: !jogoPausado significa "jogo não está pausado".

Expressões como se ((!vitoria || tempo == 0) && jogoIniciado == true) são chamadas de expressões compostas, pois combinam várias condições lógicas usando operadores como && (E) e || (OU). Sempre que usamos múltiplos operadores, o computador segue uma ordem de precedência, então é importante usar parênteses para evitar confusão.
